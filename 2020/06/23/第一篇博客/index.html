创建技术博客

发表于 2020-06-23 更新于 2020-06-24 分类于 javascript ， 数组
摘要：建立HEXO博客

前言

今天花了半天时间，终于把这个技术博客跑起来了，也遇到了一些小坑，不过多看看文档，查些资料都还是解决了。这篇文章写的很不错，官方文档讲的比较粗犷，我通过这篇文章设置了多主题，背景，外部链接等等，在此感谢作者。想过从零手写搭建，这样当然更能把这几年的所学所用，更加系统的串联起来。然而，我更清楚，自己创建博客的初衷，是为了记录和分享日常工作中所想所悟所得的干货，那些更基础，更底层，更刨根问底的原理知识，一般都是看阮一峰、张鑫旭等等老师的文章，细腻易懂且不会像官方文档那般枯燥，于是，‘拿来主义’的想法便占了上风。博客运行起来后，我更是窃喜，自己选择是对的，不然，自己利用业余时间设计，搭建，怎么也得一个月不止了。

立个FLAG

立Flag 是一件容易的事，但能坚持执行却不是每个人都能做到了。也许是因为工作很忙，加班到很晚；也许是周末了想陪陪家人或朋友，出去游玩散心。
总之，我们可以为自己不想做某件事找千百个借口，哈哈哈哈。 比如，马上就是传统节日端午节了，少不了亲戚朋友聚聚，聊聊工作生活，外出走走。
也是刚刚好，行政美女把公司发的李子柒网红粽子递到我手上后，我最后一个小问题也解决了，博客正常运行起来，也记住了几个常用的命令。以后会尽量多写写文章，安静的思考，冷静的用文字表达，在这块属于我的田野上，让自己思绪飞扬起来，不用像日常的社交那样有诸多顾虑和压力了。

结尾

前端技术因为交互终端的多样化，应用场景的多变，同时基于互联网的快速发展，催生了众多技术栈，多而杂，随便举几个例子

  • PC
  • PAD
  • android mobile
  • ios mobile
  • wechat app(当然还有支付宝小程序、字节系、京东系)
  • PWA
  • (H5 | Hybrid | Native) App
  • 数据可视化

可能我上面这句话说的不是很准确，一切还是以市场为主导的，技术是一直在突破和发展着。。。以后，可能不会仅仅只写技术文章，也会写写生活中的经历和感想吧。。
后天是2020年6月25日，农历五月初五，端午节，纪念屈原的日子。放假3天，祝所有读者和自己，工作顺利，生活顺心！
